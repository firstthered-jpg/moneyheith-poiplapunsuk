## 2️⃣ **Compare Mode** 📊

### Period Comparison
- 📅 **Month vs Month**
  - เดือนนี้ vs เดือนก่อน
  - เดือนนี้ vs เดือนเดียวกันปีที่แล้ว
  - Difference % (เพิ่ม/ลด เท่าไหร่)

- 📅 **Year vs Year**
  - ปีนี้ vs ปีที่แล้ว
  - Quarter Comparison (ไตรมาส)

- 📅 **Custom Range**
  - เลือกช่วง 2 วันที่ตรวจสอบ
  - Side-by-side view

### Metrics in Comparison
- รายได้รวม
- ค่าใช้จ่ายรวม
- กำไรสุทธิ
- Profit Margin
- Transaction Count
- Average Transaction Size

## 3️⃣ **Trend Detection** 🤖 (AI/Pattern)

### Automatic Alerts
- 📈 **Growth Detection**
  - รายได้โตต่อเนื่อง N วัน
  - Growth Rate %
  - Predicted Next Month Revenue

- 📉 **Decline Detection**
  - รายได้ตกอย่างต่อเนื่อง
  - Alert: "Revenue down 20% this week"

- ⚠️ **Anomaly Detection**
  - ค่าใช้จ่ายผิดปกติ (2x average)
  - Revenue Spike/Drop
  - Unusual Pattern

### Pattern Recognition
- 🔄 **Cyclical Patterns**
  - Weekly Cycle (วันจันทร์-อาทิตย์)
  - Monthly Cycle (ต้น/กลาง/ปลายเดือน)
  - Seasonal Cycle (ฤดูกาล)

- 📊 **Category Trends**
  - Category ไหนโตเร็ว
  - Category ไหนสม่ำเสมอ
  - Category ไหนลดลง
